5. 自由现金流（FCF）多版本 ◆

FCF 无统一定义，分三版呈现，公式全部展示。

FCF 版本 公式 FY2023 FY2024 FY2025
V1 = OCF − capex（最严口径） OCF − PP&E purchases (40,098) − 4,782 = (44,880) (58,823) − 6,771 = (65,594) (42,891) − 1,478 = (44,369)
V2 = OCF − net investing（含 JV/投资） OCF + Net investing CF (40,098) + (5,782) = (45,880) (58,823) + (9,271) = (68,094) (42,891) + (1,386) = (44,277)
V3 = OCF（无 capex 调整，fabless 近似） OCF (40,098) (58,823) (42,891)

◆ 计算得出，源数据见第 0/2 节。

  • 因 fabless、capex 极小，三版 FCF 差异不大（FY2025 约 −$42.9M 到 −$44.4M）。
  • FY2024 是 FCF 最差年（约 −$65.6M）——营收峰值年反而烧钱最多（capex $6.8M + OCF −$58.8M），印证「营收与现金创造脱钩」。
  • FY2025 FCF 改善到约 −$44.4M，但靠的是降本/SBC 反转（不可持续），且 Q1'26 已反弹（见第 7 节）。

6. 现金 runway 情景（base / bear，含 ▽ 敏感性）

起点：2025-12-31 cash & equivalents = $236.857M（零债务）。Runway = 现金 ÷ 年燃烧速率。

6.1 燃烧速率假设

情景 年燃烧（OCF + capex） 依据
Base ▽ $44.4M/年 FY2025 FCF V1（−44,369），假设维持降本后的低燃烧
Bear ▽ $65.4M/年 Q1'26 OCF(−16,362) × 4 = 65,448，假设 R&D 加码 + AP 回吐延续
Bull ▽ $30M/年 进一步降本 + 转型营收开始贡献（乐观推断）

▽ 敏感性结论：

  • 即便 Bear 情景，runway 仍有 ~3.6 年——短期无 going-concern 风险（00 档案 Agent E 印证，零债务 + $237M 现金）。
  • 但 Q1'26 燃烧已年化到 $65M（Bear），若延续，到 ~2029 需再融资；考虑到 NVTS 已展示「serial dilution」习惯（ATM 仍可继续发），届时大概率再度股权稀释。
  • 关键变量：转型营收何时贡献正现金。若 FY2026 营收继续下滑（Q1'26 年化 ~$34M），毛利仅约 $13M ▽，覆盖不了 $50M+ opex，燃烧难以下降。

8. 现金流总判（定调）

  1. 盈余质量低：GAAP 净利/亏与 OCF 大幅背离，FY2022「+$72.9M 盈利」对应 −$44.5M OCF——必须看现金流，不看 headline。
  2. fabless 轻资产：capex 极小（FY2025 仅 $1.5M），三版 FCF 差异不大，约 −$44M。
  3. 现金全靠稀释：FY2025 $194.6M 筹资流入（ATM+PIPE $200M 毛额），换来 22.5% 股权稀释；现金桥与 BS 完全 tie-out ✅。
  4. runway：base ~5.3 年 / bear ~3.6 年，短期无 going-concern 风险；但 Q1'26 燃烧已年化到 $65M，若延续到 ~2029 需再融资（大概率再稀释）。
  5. 营运资本非主因，但 AR 集中度（单一经销商占 58% 应收）+ Q1'26 存货逆势回升是两个 ⚠️ 监控点。
