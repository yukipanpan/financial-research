一句话结论

轻资产、零负债、握有 GaN+SiC 两张技术牌的小型 fabless 功率半导体公司，正用增发换来的 ~3 年现金跑道， 赌「AI 数据中心高功率供电」的未来；当前真实主营年年亏损、营收 FY2025 −45%（真实低谷 Q4'25 单季 ~$7.3M）、单一经销商占 46%（Q1'26 恶化至 59%）、TSMC 退出 GaN、 创始团队大换血 + 两年两次换审计。安全感来自零负债（不会像 Wolfspeed 那样被债务压垮），悬念在于新业务能否在现金烧完前接棒。
