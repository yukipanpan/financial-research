🔍 Phase 0.1 — 初始印象（读最新一期 + Investor Deck 后）

(a) 一句话业务骨架

Navitas Semiconductor 是一家无晶圆厂（fabless）的下一代功率半导体公司，设计销售两类宽禁带（wide-bandgap）功率器件：GaNFast™（氮化镓 GaN） 与 GeneSiC™（碳化硅 SiC），用于把电能更高效地转换/传输。历史主战场是手机快充等消费电子，自 2025 年起以「Navitas 2.0」战略转向高功率市场（AI 数据中心、电网与能源、性能计算、工业电气化）。
