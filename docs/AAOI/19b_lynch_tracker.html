AAOI × Peter Lynch 季度复查跟踪表（Quarterly Story-Validity Tracker）

Lynch 第二纪律：每季只问一个问题——"故事还成立吗？"，不看股价、不设目标价（"水草理论"：不要因为股价跌就卖）。 本表把 19_peter_lynch_analysis.md 里"故事成立的三条关键证据"做成可每季滚动更新的仪表盘。 数据来源：每期 10-Q / 10-K（★ SEC 直接披露）；比率为 ◆ 计算。不构成投资建议。 最后更新：2026-06-07 · 数据截止：FY2025 10-K + Q1-2026 10-Q

0. 三条证据 = 三个"故事成立"的必要条件

Lynch 框架下，AAOI 的多头故事要成立，必须同时看到这三件事往好的方向走。任何一条持续恶化 = 故事出洞。

# 证据 它在验证什么 ✅ 故事成立（看到这个） 🔴 故事破（看到这个）
A 经营亏损率（Op Loss / Rev） 规模效应到底跑没跑通（能否走向盈利） 逐季收窄并趋近转正 停滞在负值或再度走阔
B 应收周转天数 DSO（尤其 Digicomm） 增长是真需求还是"供应商融资"堆出来的 DSO 回落、Digicomm 账期收敛 DSO 持续拉长（周期见顶/坏账前兆）
C Amazon 权证 vesting "$40 亿大单"从故事变现金的进度 表现归属股开始 vest、Amazon 进 >10% 客户表 长期 0 vest（纯期权，故事不兑现）

Lynch 读法：从 −60% 谷底大幅收窄到 −9% 区间，方向对；但仍为负、且不平滑（25Q1 触及 −8.9% 后又回吐到 −15%，研发/管理费随营收同步膨胀吃掉了规模红利）。 当前判定：🟡 改善中但未证实——故事 A 尚未成立（需看到连续两季继续收窄并逼近 0）。

Lynch 读法：DSO 从 2024 年的 ~105–130 天结构性抬升到 156–187 天，且最大客户 Digicomm 被给予 ~304 天超长账期——这正是 Lynch 周期股的见顶/坏账前兆信号（"库存堆积 / DSO 拉长"）。增长有相当部分是用资产负债表给经销商"供应商融资"换来的。 当前判定：🔴 恶化——这是三条里最弱的一条，故事 B 不成立，需重点盯 Digicomm 账期是否收敛、是否出现坏账计提。

Lynch 读法：典型"将会成功"而非"已经成功"。$40 亿/794 万股几乎全是未兑现期权。 当前判定：🟡 未兑现——故事 C 尚未成立；关键看后续季报是否出现"第二批权证开始 vest"或 Amazon 首次进入 >10% 客户名单。

4. 综合判定（本季）

证据 方向 故事成立？
A 经营亏损率 🟡 改善但未转正、不平滑 未证实
B Digicomm DSO 🔴 恶化（180 天 / Digicomm 304 天） 否

Lynch 式结论（2026Q1）：多头故事整体尚未证实，且有一条（DSO）在恶化。按 Lynch 纪律，这不是"故事已破"（A 在改善、公司死不了、CEO 净买入），但也远未到"故事坐实"——继续放观察名单，等下季三条同时给出方向。不因股价涨跌做动作，只跟故事。
