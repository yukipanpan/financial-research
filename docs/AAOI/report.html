1. 公司一句话定义

AAOI 是一家垂直整合的光器件/光模块制造商——自产激光芯片（VCSEL / DFB / EML）并封装成光收发模块，服务数据中心、CATV（有线电视）、电信、FTTH 四大终端市场。NASDAQ 上市，创始人 Dr. Thompson Lin 兼任董事长+总裁+CEO。★ FY2025 10-K Item 1

核心术语三件套（节选）：

  • 光模块（Optical Transceiver）：把电信号↔︎光信号互转的"小盒子"，数据中心/电信网络的数据管道接口。AAOI 的主营产品。★
  • 垂直整合（Vertical Integration）：AAOI 自己造激光芯片（die）再封装成模块，而多数同行只做模块组装、芯片靠外购——这是 AAOI 唯一的结构性差异化。★ FY2025 10-K Item 1

  • contra-revenue（冲减收入）：给客户的股权激励（Amazon 认股权证）按会计准则从收入里扣减，而非计费用。★ FY2025 10-K
  • 议价权判断：偏弱——尽管垂直整合，但下游客户极度集中（前五 95.2%、Digicomm 一家 53%），上游 DSP 仍依赖 Marvell/Broadcom。▽

10. 会计政策深度（详见 05_accounting_policies.md）

  • 收入确认：ASC 606，出货时点确认（point-in-time），无 over-time 估计 → 操纵空间小（🟢）。★
  • 折旧：直线法；厂房 20–42 年、机器设备 2–20 年、计算机 3–10 年。机器设备 20 年上限约为同业（COHR/LITE/FN 设备多 ~10 年）的 2× → 长端偏激进（+1）。★
  • 资本化：R&D 全部费用化（🟢）；无利息资本化（政策允许但选择不用，🟢偏保守）。★ Note H
  • 存货：平均成本法，孰低计价。★
  • Amazon 认股权证：权益分类、列 contra-revenue（非负债/衍生品）；已归属部分记 16.0million资产（1.0M 流动 + 15.0M非流动），FY2025仅冲减收入 0.8 million（immaterial）。★
  • 激进度综合评分：−2（略偏保守）。11 项里保守占优（R&D 费用化、零利息资本化、出货确认、无商誉）；仅"机器设备 20 年上限"和"$39.8M 在建工程不减值判断"两项 +1 偏激进。
  • 最大灰色地带：在持续经营亏损 + AI 产能豪赌下对在建工程的"不减值"判断，依赖乐观的未来现金流假设。

18. 我的论点（含反方）

论点 1：这是"双周期 + 双巨头绑定"驱动的真实营收拐点，但盈利证明仍在前方。

  • 证据：营收 2024Q1 谷底 $40.7M → 2026Q1 $151.1M（3.7×）每季上行；毛利率永久重定级到 ~30%；Amazon $4.0 billion 框架 + Microsoft 5 年 SOW。
  • 反方：增长靠极少数客户 + CATV 周期；8 年经营亏损、R&D/G&A 暴涨吃掉规模红利。
  • 反驳：拐点真实，但需见经营亏损率持续收窄/转正 + Digicomm 外客户多元化才算"质变"。

论点 2：现金消耗性质被市场误读——是营运资金扩张，不是经营失血。

  • 证据：FY2025 OCF −174M中 228M 是应收+存货建仓；净亏被非现金项抵消，经营近平衡。
  • 双刃：好处是经营不烧钱；坏处是现金风险压在一个经销商应收上，且需持续融资周转。

论点 3：估值的真正特殊性 = 两大科技巨头的战略锁定。

  • Microsoft 收购优先权（ROFR）+ Amazon $4.0 billion 绑定权证 → AAOI 既是供应商也是潜在并购标的，是个"期权"，但命运高度系于两家。

论点 4：增长由稀释买单。

  • 累计融资 ≈$840M、完全摊薄 92M 股、悬顶 16.7%。每股价值能否提升取决于剔除营运资金后的经营现金流何时真正转正。

19. 悬而未决问题 → 已深挖解决（详见 06_capital_structure.md / 04b_cash_flow_DEEP.md / 15_operating_asset_deep_dive.md）

① Digicomm 应收账期/坏账 → 解决：是"供应商融资"，坏账准备几近为零但 DSO 极长。

  • 坏账准备金额可忽略（FY2024 仅 $1–3 thousand；FY2025 坏账费用 $4 thousand）→ 账面无减值。★ Q1-26 10-Q Note 9
  • 但 Digicomm $222.7 million = 74.5% 应收，管理层明示给予 "longer than typical payment terms"（让其囤 CATV 备货）。
  • ◆ 混合 DSO = 299.0 / 151.144 × 91 = 180 天；其中 Digicomm 单独 ~304 天（≈3.3 个季度的 Digicomm 收入），其余客户 ~82 天。实质是把 vendor financing 记成普通应收——单点流动性/信用红旗。

② 2030 Notes 完整契约 → 解决：$125.0 million、2.750%、2030-01-15 到期、转股价 43.31（23.0884股/1,000）。 ★ FY25 10-K Note L

  • 发行人赎回权自 2027-01-15 起，仅当股价 ≥ 转股价 130%（≈$56.30）于 30 个交易日中 20 日成立。
  • 无 capped call / bond hedge → 转股稀释无对冲（全额暴露）；契约 covenant-light（无杠杆率/流动性数值阈值）；含特殊的 "Specified Divestiture" 或有赎回条款。
  • 账面 $129.5 million vs 公允价值 $278.7 million（股价冲破转股价后），近 18 个月无重大到期（🟢）。BOK $35 million 循环额度限制债务/留置/并购但无数值杠杆 covenant。

③ 800G/1.6T 出货 + Amazon vesting → 解决：800G 正在放量，但 Amazon 几乎还没真正买。

  • 产品代际：400G 是当前主力，★ 10-K 称 "800G 很可能在 2026 下半年超过 400G 销售"，1.6T 仍在客户测试/认证。800G = AI 互联产品。设计导入 6→8→9（2023→2025）。
  • Amazon 认股权证：6,621,166 股表现归属部分至今 0 股 vested（仅初始 1,324,233 股）。★ Amazon 累计采购 ▽ ~71million（FY25 33.1M + Q1-26 ~$38M），不足 4.0billion触发线的240 亿/794 万股 = 几乎全是尚未兑现的期权。

④ 美国新厂折旧/产能利用 → 解决：在建工程是未来利润拖累。

  • 在建工程 CIP $39.8 million（FY25）→ $51.5 million（Q1-26），尚未计提折旧（约 6.5% 的总 PP&E 是未来才进 P&L 的折旧）。折旧费用已从 20.2M→27.2M（FY25）抬升。毛利率回升（24.8%→30.0%）主因 CATV 放量改善固定成本吸收。★

⑤ PSU 现金结算风险 → 已消除（2026-06-04 股东大会）。 ★ 2026-06-05 8-K Item 5.07

  • 风险原值：若达 200% 业绩且 2026 股权计划未获批，需现金结算 $64,872,156（766,901 份 PSU，5 名高管）。
  • 结果：2026 股权激励计划已获股东批准（赞成 34,862,167 vs 反对 3,780,336），2026-06-04 生效，取代旧 2013/2021 计划。→ PSU 现可用股票结算，~$64.87M 现金流出风险解除。同次大会 PwC 经投票正式获批为 FY2026 审计师、say-on-pay 通过、章程修订通过、两名董事（Che-Wei Lin、Robert Flanagan）当选。
