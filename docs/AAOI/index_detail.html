16 号不触发说明：AAOI 无加密/数字资产财库、无证券组合、无权益法投资、无停止经营。唯一的"盈余扰动项"是 FY2024 的一次性债务清偿损失 $111.962 million 与 Amazon 认股权证 contra-revenue——已在 14 号档案 + 报告 §13 处理，无需单独开 16。

Phase 0.1 初始印象（最新一期 Q1-2026 10-Q + FY2025 10-K）

  • 一句话业务：垂直整合光器件/光模块制造商（自产激光芯片→模块），四终端：数据中心、CATV、电信、FTTH。
  • 管理层叙事锚：营收同比高增（+51% YoY Q1-26）、毛利率守住 ~30%、数据中心 AI 需求、Amazon/Microsoft 大客户。
  • 口径警觉：强调营收增速与"top customers"，弱化经营仍亏损、应收暴增（Digicomm 一家 $222.7M）、净亏被一次性项扰动。
  • 待验证假设 → 结论：
    1. "AI 数据中心是增长引擎" → ⚠️部分推翻：2025 全年 CATV(Digicomm) 才是 #1（53.8%），数据中心 2026Q1 才夺回。
    2. "营收翻倍→快要盈利" → ❌推翻：8 年连续经营亏损，规模杠杆未跑通。
    3. "净亏大幅收窄=经营改善" → ❌推翻：FY2024 巨亏含一次性 111.962M，剔除后经营基础亏损 −74.7M。
    4. "靠自身造血扩张" → ❌推翻：三年累计 FCF −484M，全靠融资（累计≈840M）。
