1. 公司一句话定义

Digi Power X Inc.（前身 Digihost Technology Inc.，2025-03-06 更名 ★10-K）是一家从比特币矿工转型为「电力驱动的 AI/HPC 数据中心 + 托管」运营商的能源基建公司：自有一座 60MW 燃气联合循环电厂，控制约 197MW 电力容量，正用自研 ARMS 模块化机柜（搭载 NVIDIA B200/300）把矿场改造成 Tier III AI 算力中心。双重上市：Nasdaq DGXX + Cboe Canada DGX。

6. 收入构成与客户集中度

  • 客户集中度（红旗 ✅ 存在，但未量化披露）：filings 未给 「X% 收入来自客户 Y」的量化集中度表（披露缺口 ⚠️）。但定性上 colocation/售电由少数几份合同构成 ★[10-K]。
  • Northern Data NY, LLC 是「售电」的唯一客户（≥22MW、24/7）★[20-F]，贡献 FY2024 $6.28M、FY2023 3.04M，2024年合同到期后FY2025归零—— * *单一客户流失直接抹掉 6.3M 收入**。

  • 5W 检验：WHAT=电力+机柜+场地托管；WHO=Northern Data（已走）+ 数家未具名矿企；HOW MUCH=按实际用电量月结；WHEN=over-time 月度确认；WHY=自有低价电力(~$0.04/kWh)。

18. 我的论点（含反方观点）

主论点 1 — reported 巨亏被严重误读；真实经营亏损稳定但仍未盈利。

  • 证据：① 剥离测试 FY2025 经营亏损 ~17.6M ≈ FY2024 17.8M（§13）；② reported $28.4M 的恶化几乎全是加密/FX/权证 mark-to-market；③ 会计政策偏保守（§10）。
  • 反方：调整后仍亏 $17.6M，且含 $8M SBC；毛利仍为负（毛亏 $3.2M）；加密浮亏 $3.76M 是真实价值缩水。
  • 我的反驳：论点是「reported 数字误导」，不是「公司健康」。两者都成立——经营未盈利，但没有 reported 显示的「断崖式恶化」。

主论点 2 — 这是一个「有电、有机柜、没客户」的转型期权(option)，价值与风险都在尚未签约的 AI 租户上。

  • 证据：① 设备押金真金投入 0→20.9M；② 但 AI 分部 FY2025 收入 $0；③ 下游 AI 客户在 deck 中是「目标」而非签约合同（§4 产业链下游空缺）。
  • 反方：~197MW 自有低价电力 + Tier 3 认证 + $78M 现金 + 零债务，是真实可变现的转型底座。
  • 我的反驳：底座真实，但估值/前景完全取决于能否把电力容量转成签约 AI 收入——这一步尚未发生，是纯期权。

主论点 3 — 资产负债表干净，但靠持续稀释维系；治理集中是结构性折价项。

  • 证据：零债务 + $78M 现金（§8）；但 SV 股 3 年增至 3.1 倍 + ATM $200M（§14）；Amar 父子控盘 + 双层股权 + 高管薪酬 3.8x 暴增 + BTC 计酬（§16）。
  • 反方：零债务使其在 crypto/AI 寒冬中比高杠杆同业更抗打。
  • 我的反驳：抗打成立，但代价是现有股东被持续稀释，且治理集中度高——是「干净的资产负债表 + 不便宜的治理」。
