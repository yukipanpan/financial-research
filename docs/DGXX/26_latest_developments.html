26.1 2026-06-03 8-K（Item 7.01 + 9.01）★ — Vera Rubin 采购 + 运营/财务更新

(a) $35M NVIDIA Vera Rubin 采购承诺 ★

  • 承诺 $35M 购买 NVIDIA Vera Rubin 系统——Blackwell 的 rack-scale 下一代平台：Rubin GPU（288GB HBM4）+ 88 核 Vera CPU + NVLink 6
  • 初始部署目标 2027-Q1（取决于 NVIDIA 产能/交付）
  • 用自有现金支付，通过 NeoCloudz GPU 即服务平台出租
  • 叠加此前 20MB300（2026 − 01） → NeoCloudz自营GPU机队累计承诺采购 * * 55M**

(b) 财务状况更新（截至 2026-06-03，◇ 管理层）

  • 现金及等价物 ~$150M（对比 Q1-2026 末 $57.8M ★、FY2025 末 78.5M★）——印证Q1后ATM继续大额募资（10 − Q后续事件披露的 102.86M ATM 增发 ★ 是主要桥梁）
  • YTD 已向阿拉巴马站投入 ~$65M 资本开支，全部自筹（对应 Q1-2026 设备押金 $20.9M + 后续投入）
  • 正在敲定**项目级融资（project-level financing）**以支持下一阶段（即 Cerebras Phase 2）

(c) 项目进度（重申）★

  • Cerebras 园区 Phase 1（15MW IT 负载）目标 2026-12-15 RFS；全 40MW 目标 2027-Q1
  • ⚠️ Phase 2 部署明确以"获得充分融资"为条件——融资风险被再次点名
