三个最关键的发现

1. reported 巨亏被误读。 FY2025 净亏 reported $28.4M（看似比 FY2024 翻倍恶化），但剥离加密/汇兑/权证三项 mark-to-market 后，真实经营亏损 ~$17.6M ≈ FY2024 ~$17.8M（基本持平）。

2. 业务已转型，且 AI 已签约（v1.2 更正）。 挖矿占收入 100%(FY22)→10%(FY25)；现 colocation 51% + 售能 39% 为主。AI 已签 Cerebras（40MW / $1.1B起 / 10年）+ SubQ（$19.6M/24月），但 Cerebras Phase1 要 2026-12 才交付——从「没客户」变成「已签约、待交付」。

3. 干净资产负债表 + 靠稀释维系。 零计息债务、$78M 现金，但全靠 $104.7M 增发 + ATM($200M)；SV 股一年翻倍、三年增至 3.1 倍。
